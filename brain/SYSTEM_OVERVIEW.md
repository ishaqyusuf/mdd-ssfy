# System Overview

## Purpose
Quick orientation for contributors and AI agents working in the GND monorepo.

## Summary
- Monorepo managed with Turborepo and Bun workspaces.
- Primary applications live under `apps/` for web, API, mobile, and supporting surfaces.
- Shared business logic and infrastructure live under `packages/`.
- Project documentation and execution memory live under `brain/`.

## Start Here
1. Read `brain/BRAIN.md`.
2. Read `brain/system/overview.md` and `brain/system/architecture.md`.
3. Read `brain/tasks/in-progress.md` and `brain/progress.md`.
4. Open any related `brain/features/*.md` or `brain/decisions/*.md` files before changing behavior.
