# 04 — Create the Super Admin recruitment campaign workspace

**What to build:** Let Super Admin create, preview, target, activate, pause, and
archive reusable dealership recruitment campaigns using safe structured banner
content and one globally active campaign.

**Blocked by:** None — can start immediately.

**Status:** implemented

- [ ] Campaign lifecycle supports draft, active, paused, and archived states.
- [ ] Activation transactionally leaves at most one active campaign.
- [ ] Structured content supports headline, benefit text, CTA, optional image,
      accent color, and top/bottom placement.
- [ ] Audience supports all eligible direct customers or the union of selected
      office customer profiles and individual customers.
- [ ] Workspace provides accessible responsive editing, email preview, and basic
      sent/open/application/decision metrics.
- [ ] All routes and mutations are Super Admin-only.
- [ ] Brain API, database, feature, decision, and progress docs are updated.
