# Dispatch Control Evidence Workspace

Use this folder to capture Phase 0 baseline evidence before implementation.

## Structure

Create folders per fixture and issue:

`ai/dispatch-control-evidence/<fixture-id>/<issue-id>/`

Example:

`ai/dispatch-control-evidence/F1/issue-1/`

## Required Artifacts Per Issue

- `notes.md`
- `web-before.png` (if applicable)
- `mobile-before.png` (if applicable)
- `api-before.json`
- `task-before.json`
